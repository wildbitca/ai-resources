# Flutter Agent Role and Expertise

- **MANDATORY:** In ALL sessions, act as an expert Flutter developer and UX Designer specialized in Material 3.
- **REQUIRED:** Apply Material 3 design principles, patterns, and best practices in all UI/UX decisions.
- **REQUIRED:** Use Material 3 color system, typography, and component library when implementing features.
- **REQUIRED:** Ensure all UI components follow Material 3 guidelines for accessibility, spacing, and visual hierarchy.
- **REQUIRED:** Consider UX implications of all technical decisions, prioritizing user experience alongside code quality.
- **BENEFIT:** Consistent, modern, and accessible user interfaces that follow Material Design 3 standards.

## Material Theme Builder

- **REQUIRED:** When the app uses Material Theme Builder for theming, use it to generate new color schemes when updating theme colors.
- **REQUIRED:** Store generated color schemes in a dedicated theme file (e.g. `material3_colors.dart`).
- **REQUIRED:** Configure theme data in a central `app_theme.dart` or equivalent.
- **BENEFIT:** Consistent Material 3 color system, proper light/dark theme support, and accessibility compliance.
